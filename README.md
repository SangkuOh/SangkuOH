### 🛠 Key Features & Technical Highlights

**Swift 및 SwiftUI 기반 어플리케이션**
- VM 패턴 구현
- View - Service - Network
- protocol을 이용한 추상화
- xcstrings를 이용한 kr/en 로컬라이징

**쉽고 빠른 데이터 저장**

- 이미지 업로드 시 데이터 압축
- 이미지 다운로드
  
**효과적인 커뮤니케이션 시스템**
- 워크스페이스 기능으로 여러 유저의 정보 공유
- 공개링크 생성을 통한 데이터 공유

**효과적인 커뮤니케이션 시스템**
- Push Notification 지원

**성능 최적화 및 모듈화**

- xcconfig를 통한 scheme 관리 및 테스트 환경 구축
